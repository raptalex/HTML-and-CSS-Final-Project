Favorite Programming Language

I quite like the C++ programming language. It's with the "C Family" of programming languages which have syntax that I quite like because of how precise it is.

C++ is by no means a perfect language and it can be cumbersome to learn how to create programs in it.

This is largely offset by the amount of libraries made in the language to accomplish a variety of tasks and the overall design being well rounded for many kinds of tasks.

Since it is also a compiled language, it is quite fast, which is a very important upside of the language.
